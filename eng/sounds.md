# Sounds

General sounds
-

- **bang** = грохот, выстрел
- **slurp** = хлебать с шумом
- **tick** = тикать (часы)
- **click** = щелчок
- **buzz** = жужжание
- **tap** = постукивать
- **splash** = всплеск
- **sniff** = шмыгать носом
- **whistle** = свистеть, свисток
- **crash** = авария, грохот (падения, удара)
- **snore** = храпеть
- **crunch** = хруст
- **creak** = скрип
- **drip** = капать
- **hiss** = шипеть
- **hoot** = сигналить (клаксоном), уханье (совы)
- **hum** = напевать, гудеть
- **rattle** = дребезжать, греметь
- **roar** = рёв, гул
- **screech** = визг (тормозов, птицы)
- **slam** = хлопать (дверью)

Human sounds
-

- **scream** = орать
- **yell** = кричать, вопить (обычно от злости/для внимания)
- **giggle** = to laugh in a silly way = хикать глуповато или смущённо
- **whisper** = to speak very quietly = шептать
- **mumble** = бормотать, неразборчиво говорить
- **groan** = to make a long deep sound because you are in pain or annoyed = стонать
- **stammer** = to speak with difficulty, often repeating sounds or words = заикаться, запинаться
- **sob** = рыдать
- **sigh** = to let out a long deep breath = вздыхать
