Модель Хольта-Уинтерса
-
Предполагаем, что веменной ряд состоит из
- Уровня
- Тренда
- Сезонности

Для каждого пункта есть гиперпараметр из $(0,1)$:
- $\alpha$ - для уровня
- $\beta$ — для тренда
- $\gamma$ — для сезонности

$L_t = \alpha(Y_t - S_{t-s}) + (1-\alpha)(L_{t-1} + T_{t-1})$

$T_t = \beta(L_t-L_{t-1}) + (1-\beta) T_{t-1}$

$S_t = \gamma\frac{Y_t}{L_t} + (1-\gamma)S_{t-s}$

$F_{t+k} = L_t + kT_t + S_{t+k-s}$
